# AI Resume Builder

A professional resume builder with **14+ resume templates**. Browse, preview, select, and create your resume with a live editor.

## Features

- **14+ Professional Templates** — Classic, Modern, Minimal, Professional, Creative, Executive, Technical, Elegant, Compact, Two Column, Bold, Academic, Startup, and Timeline
- **Template Gallery** — Visual preview cards with sample data so you can compare designs
- **Full Preview Modal** — Click any template to see a full-size preview before choosing
- **Search & Filter** — Find templates by name, category, or tags
- **Live Editor** — Fill in your details and see changes instantly
- **Auto-save** — Your progress is saved locally in the browser
- **Download PDF** — Print or save as PDF using the browser print dialog

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## How to Use

1. Browse the template gallery on the home page
2. Click **Preview** to see a full-size sample resume
3. Click **Use Template** to start building your resume
4. Fill in your personal info, experience, education, and skills
5. Click **Download PDF** when you're ready
